英语句子一般有三种层次: 
  • 简单句
    • We are sitting here in the sun.
    We can see snow-covered hills.

  • 复杂句
    • We are sitting here in the sun, 
    对等子句
    and we can see snow-covered hills.
    对等子句

    As we sitting here in the sun,
    从属子句
      we can see snow-covered hills.
    主句

  • 简化子句
    • Sitting here in the sun,
    非限定子句
     we can see snow-covered hills.
    限定从句

    简单句是句子的最基础的组成成分.一个简单句必不可少的部分是主语和谓语.
    • 主语: 句子的主角, 只能是名词类.

    • 谓语: 带领句子的其余部分来叙述主角做了什么事或者是什么样, 一个句子中必须有一个限定动词.

    • 宾语: 动作的接收方, 只能是名词类
    限定动词:
    即传统语法中的动词, 而非限定动词在语法中不视为动词.

    e.g.I can read this book
     这句话中can是非限定动词, read是限定动词.

    限定动词有两种属性:
    1. 及物/不及物:
      2. 及物动词后面必须要有宾语来接收动作. 不及物可以独立表示动作不需要接受方.
    2. 完全/不完全:
      3. 不完全动词需要补语说明动作, 完全动词则不需要
